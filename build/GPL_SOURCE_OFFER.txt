OFFER FOR SOURCE CODE - GPL-COVERED COMPONENTS
==============================================

GKMacOpener binary distributions may include object-code forms of GPL-covered
components, including:

- a custom WSL2 Linux kernel binary: wsl_kernel
- Linux filesystem kernel modules: hfsplus.ko, hfs.ko, apfs.ko
- optional APFS-FUSE-related binaries, if a particular release explicitly
  includes them

Recipients of GPL-covered binaries are entitled to the complete corresponding
source code for those GPL-covered portions under the terms of the applicable
GPL license.

How to Obtain Source
--------------------

The public repository includes a GPL source manifest at:

docs/GPL_SOURCE_MANIFEST.md

The official GKMacOpener release notes or repository documentation must also
record the exact upstream source revisions, configuration, patches, and build
scripts used for every GPL-covered binary included in that release.

If the corresponding source is not already published in the GKMacOpener
repository, request it through the support or issue tracker published with the
official GKMacOpener repository. The project may fulfill the request by
providing a durable download link, source archive, or public repository.

Any fee charged for physical distribution will not exceed the reasonable cost
of providing the source.

Duration
--------

This offer is valid for at least three years from distribution of the
corresponding binary, and for as long as GKMacOpener offers support for that
release, whichever is longer.
